# 🌟 Legendary Tooltips x Equipment Compare 📊

This mod fixes tooltip rendering issues when Legendary Tooltips and Equipment Compare are installed by ensuring that the stack count is always hidden behind the tooltips.

## 🔄 Versions

Minecraft >= 1.21.1 <br />
Fabric >= 0.16.4 <br />
NeoForge >= 21.1.233

## ⬇️ Installation

1. Download the mod from [Modrinth](https://modrinth.com/project/gCfAwaai) or [CurseForge](https://www.curseforge.com/minecraft/mc-mods/lt-x-ec)
2. Place it into your `mods` directory

## 📦 Modpack

Feel free to include LT x EC in modpacks.
